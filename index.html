Typhoon Haima Municipality Priority Overview

This dashboard presents a priority ranking of muncipalities hit by Haima. It aims to contribute to the question which municipalities to prioritize given limited resources. This is done both on severity data (houses damaged) and on pre-disaster vulnerability indicators such as poverty. The preselection made by PRC is taken as a given, so only those municipalities are included.
