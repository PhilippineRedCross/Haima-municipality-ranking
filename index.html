Typhoon Haima Municipality Priority Overview

This dashboard presents a predictive priority index (5=high, 1=low) of muncipalities hit by Haima. Thereby it aims to contribute to the question which municipalities to prioritize given limited resources. The index is constructed by combining both severity data (houses damaged) and pre-disaster vulnerability indicators such as poverty. Note that the preselection made by PRC is taken as a given, so only those municipalities are included here.
